### Installation Guide

1. Download the mod file `.jar` matching your Minecraft version from one of the links above.
2. Make sure you have the correct Mod Loader installed, the mod uses **NeoForge** only.
3. Drop the downloaded `.jar` file into your Minecraft `mods` folder.
4. Launch the game and play!
